# 🌐 Online Multiplayer Guide

## How to Play Online

### Method 1: Create a Room (Host)
1. Click **"Online"** card on the main menu
2. Click **"Create Room"**
3. A unique 6-character room code will be generated (e.g., `A3K7NP`)
4. **Share this code** with your friend
5. Wait for your friend to join
6. Game starts automatically when they connect!

### Method 2: Join a Room (Guest)
1. Click **"Online"** card on the main menu
2. Click **"Join Room"**
3. Enter the 6-character room code your friend gave you
4. Click **"Join"**
5. Game starts automatically when you connect!

## Important Notes

### ✅ How It Works
- Uses **WebRTC** (PeerJS) for direct peer-to-peer connections
- **No server required** - players connect directly to each other
- Works across **different devices** and **different networks**

### ⚠️ Requirements
- Both players need to have the game **open in their browser**
- Modern web browser (Chrome, Firefox, Edge, Safari)
- Internet connection

### 🎮 Game Rules
- **Host** plays as ❌ (X) - goes first
- **Guest** plays as ⭕ (O) - goes second
- All moves are synced in real-time
- If either player disconnects, the game pauses

### 🔧 Troubleshooting

**"Failed to connect to room"**
- Double-check the room code is correct
- Make sure the host still has the game open
- Try refreshing both pages and creating a new room

**"Opponent disconnected"**
- One player may have closed the tab or lost connection
- Host can create a new room and share the new code

**Can't hear sound?**
- Click anywhere on the page to enable audio
- Check the speaker icon in the top-right corner
